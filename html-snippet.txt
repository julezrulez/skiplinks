Put some id's in the right place in your code to skip to, i.e.:

<div id="header">
</div>
<div id="menu">
</div>
<div id="content">
</div>


Or, if you use HTML4 or XHTML you can use:

<a name="menu"></a>
<a name="content"></a>



Please note that Chrome has some trouble skipping to a non-focusable element.
A work-around is tabindex="-1":

<div id="menu" tabindex="-1">
